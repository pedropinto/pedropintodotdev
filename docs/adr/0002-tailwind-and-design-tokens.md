# 0002 - Tailwind and Design Tokens
- Status: Accepted
- Date: 2026-02-21

## Context
The UI should remain tidy and consistent while moving quickly.

## Decision
Use Tailwind CSS with a constrained token system and the selected palette.

## Consequences
- Styling changes are fast and consistent.
- Reduced visual drift across pages.
- Requires discipline to avoid utility-class sprawl.

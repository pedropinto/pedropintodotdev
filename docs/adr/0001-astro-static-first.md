# 0001 - Astro Static-First
- Status: Accepted
- Date: 2026-02-21

## Context
The project is a minimalist personal blog focused on content readability and low operational overhead.

## Decision
Use Astro as a static-first framework.

## Consequences
- Fast pages by default and low JS payload.
- Simple hosting model (static output).
- Keep interactive features intentional and limited.
